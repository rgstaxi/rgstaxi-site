
RGS TAXI - INSTALLATION GITHUB PAGES

1. Crée un dossier local avec ces fichiers :
- index.html
- taxi-aeroport-orly-roissy.html
- taxi-gares.html
- contact.html
- styles.css
- script.js
- logo-rgs-taxi.webp

2. Sur GitHub :
- New repository
- Nom conseillé : rgstaxi-site
- Public
- Create repository

3. Upload :
- Add file
- Upload files
- Glisse tous les fichiers ci-dessus
- Commit changes

4. Active GitHub Pages :
- Settings
- Pages
- Source = Deploy from a branch
- Branch = main
- Folder = /root
- Save

5. Attends l’URL GitHub Pages.
Exemple :
https://tonpseudo.github.io/rgstaxi-site/

6. Plus tard, si tu mets un vrai domaine :
- remplace les balises canonical par ton vrai domaine
- ajoute ton domaine personnalisé dans Settings > Pages

IMPORTANT :
- garde exactement les noms de fichiers
- mets bien ton logo sous le nom : logo-rgs-taxi.webp
- si tu n’as pas encore de domaine perso, les canonical peuvent être temporairement laissés comme ça, mais l’idéal est de les adapter après mise en ligne
